## Caveman + Ponytail


Instalación: poner la carpeta .cursor/rules en la raiz del proyecto donde quiero que funcione, luego preguntarle a la IA si están activos



# Proposito:

El propósito de este repo es tenener caveman + ponytail listos para implementar en cualquier proyecto, con el objeto de reducir el consumo de tokens tanto para el mensaje que se envía y recibe de texto (caveman), como reducir el gasto de tokens en programación (ponytail), básicamente aplican un concepto similar pero uno es para el texto y el otro para el código
